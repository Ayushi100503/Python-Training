# Python Training exercises

This repository contains Python training files created during learning and practice sessions.

## Structure

- Top level folders represent the Month
- Inside each Month folder are subfolders named by the *date*(`DD.MM.YYYY`) on which the training was done
- Inside each date folder are one or more Python files covering concepts practiced that day and docx files containing explanations of the tech stacks used.
- Dedicated folders exist for **capstone project** and **final project**

**Note**: Final Project has 3 case studys
